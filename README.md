# B02_DetectiveChoiHyuk

## #Pre
원래 있던 레파지토리는 배포 금지 에셋을 포함했기 때문에 코드만 올려놓은 레파지토리를 따로 생성.
 
## #1: 팀원소개

-   박인수(팀장님)
    -   플레이어 컨디션, 3층 퍼즐, 체크포인트
-   박관엽
    -   아이템과 인벤토리, 4층 퍼즐
-   김서영
    -   아이템 상호작용, 2층A 퍼즐
-   유예린
    -   플레이어 이동 및 시야 컨트롤, 2층B 퍼즐
